Week 2

Published on October 17th, 2023

This week I set up git with my website. What git will do is keep track of all the different versions of my files. This means that if I make a change and it breaks my website I can always go back to a previous version where my website works. I have found git very useful in the past when I was making an etch a sketch website and the page would freeze after using it for a while. I had to go back to a previous version where the page wouldn't freeze and figure out the issue from there.

Published on October 17th, 2023

this week I added a table to my learning journal that showed the time schedule I followed when creating my website. I had created a table previously but I still had to do a recap over it since I had forgotten parts. This time I had to learn how to make my table responsive, I did this by following a youtube tutorial by Kevin Powell. This puts a horizontal scroll bar on the table, if I didn't do this there would be a horizontal scrollbar on the whole page which isn't ideal. Since it's a small table I thought that this solution was good enough and that it wasn't worth the effort of creating a table without a horizontal scroll bar.
