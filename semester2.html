What is meant by compiled and interpreted languages?

Compiled languages are converted into machine code then ran by the CPU whereas interpreted languages are ran line by line without the need for them to be compiled. Javascript is an interpreted language.

What is meant by weakly and strongly typed languages

Strongly typed languages have stricter typing rules which means it is more likely to run into errors when the code is ran or compiled. Weakly typed languages have looser rules which can produce unpredictable results. Javascript is weakly typed.

Three common uses for JavaScript in web development

JavaScript can be used to provide extra functionality to parts of a webpage. For example if a user clicks on text you can make that text change colour or run an animation.


Another use for JavaScript in web development is adding more html to a page in real time. This is useful for example if a user fills out a form and you want to display the results of that form on the page you can add that to the page using JavaScript.


Another use of JavaScript in web development is to connect to servers
